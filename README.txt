Привет! Это подробная инструкция по запуску проекта магазина систем видеонаблюдения на твоем компьютере. Для начала тебе понадобится установленный локальный сервер (например, XAMPP) с базой данных MySQL и актуальная версия Python. Распакуй этот архив в любую удобную папку на своем компьютере, но пока не запускай никакие файлы.

Сначала мы настроим базу данных. Открой панель управления локальным сервером и запусти MySQL (в XAMPP для этого нужно нажать Start напротив MySQL и Apache). Открой панель phpMyAdmin в браузере. В левом меню нажми Создать БД, введи точное имя cctv_db, в выпадающем списке выбери кодировку utf8mb4_unicode_ci и нажми Создать. Затем выбери созданную базу слева, перейди во вкладку Импорт в верхнем меню, через кнопку выбора файла найди файл базы данных cctv_db.sql из распакованной папки проекта и нажми Вперед в самом низу страницы. Твоя база данных вместе с добавленными товарами успешно загружена.

Теперь подготовим программную часть проекта. Открой командную строку или терминал PowerShell прямо в папке с распакованным проектом. Создай виртуальное окружение командой python -m venv venv и активируй его командой venv\Scripts\activate (если PowerShell выдаст ошибку безопасности, используй команду .\venv\Scripts\activate). После того как в терминале появится префикс окружения, установи все необходимые библиотеки командой pip install django pymysql python-dotenv Pillow. Дождись окончания загрузки и установки всех пакетов.

Осталось только запустить сам сайт. В этом же терминале введи команду python manage.py runserver. В консоли появится служебная информация и ссылка http://127.0.0.1:8000/, по которой будет доступен твой магазин видеонаблюдения. Открой эту ссылку в браузере, чтобы перейти на витрину. Для доступа в панель управления товарами перейди по адресу http://127.0.0.1:8000/admin и используй логин admin с паролем [ВПИШИ СЮДА СВОЙ ПАРОЛЬ ОТ АДМИНКИ]. Приятного использования!
